Признак Лейбница сходимости знакочередующегося ряда


Числовой ряд ∑∞k=1 ak называется знакочередующимся, если знаки любых двух соседних слагаемых противоположны: sign(ak+1) = -sign(ak)


Пример 1

1 - 1/2 + 1/3 - 1/4 + ... + (-1)n-11/n + ...


Если слагаемые знакочередующегося ряда по модулю монотонно стремятся к 0, то ряд сходится:

|a1| ≥ |a2| ≥ ... ≥ ak ≥ ... → 0


Замечание 1. Доказательство этой теоремы, как и многих признаков сходимости числовых рядов, опирается на теорему о существовании предела у монотонной ограниченной последовательности (частичных сумм ряда). Для знакоположительных/знакоотрицательных числовых рядов монотонное стремление к нулю слагаемых не является гарантией сходимости (частичные суммы могут быть неограниченными), но за счѐт чередования знаков слагаемых сходимость достигается.


Замечание 2. Ясно, что для сходимости знакочередующегося ряда достаточно, чтобы условие теоремы стало выполняться, начиная с некоторого номера n0.


Пример 1 (продолжение) Исследовать на сходимость

1 - 1/2 + 1/3 - 1/4 + ... + (-1)n-11/n + ...

Данный ряд носит название ряда Лейбница.


Решение. Это знакочередующийся ряд. Модули его членов убывают:

1 > 1/2 > 1/3 > ... > 1/n > ...

и предел их равен нулю; по признаку Лейбница ряд сходится. (Лейбниц показал, что значение (сумма) этого ряда равно ln2)


В примерах 2 и 3 для вычисления предела последовательности можно использовать соотношение:

limn→∞ n/an = 0 при a > 1


Пример 2 Исследовать сходимость ряда

∑∞n=0 (3n-1)/(-5)n

Решение. Это знакочередующийся ряд. Сначала докажем, что |an| ≥ |an+1|


Способ 2 Рассмотрим производную функции

f(x) = (3x-1)/5x ⇒ f'(x) = (3*5x - (3x - 1)*5x * ln5)/52x

Знаменатель > 0, исследуем числитель: 3*5x - (3x - 1)*5x * ln5.

1 < ln5 < 2, поэтому ln5 - 3 * (x - 1) < 0 при x ≥ 2

Функция убывает, значит, модули слагаемых ряда убывают.
Условие f(x) → 0 при x → ∞, очевидно, выполняется (можно применить правило Лопиталя). Ряд сходится по признаку Лейбница.

Способ 1

un = (3n - 1)/5n; un+1 = (3(n + 1) - 1)/5n+1 = (3n + 2)/5n+1

Рассмотрим:

un - un+1 = (3n - 1)/5n - (3n + 2)/5n+1 = (15n - 5 - 3n - 2)/5n+1 = (12n - 7)/5n+1


Для всех n ≥ 1 выполняется неравенство 12n − 7 > 0. то есть члены ряда по модулю убывают. Найдѐм предел модуля общего члена ряда (применяя правило Лопиталя???):

limn→∞ |(3n - 1)/(-5)n| = limn→∞ (3n - 1)/5n = ∞/∞ = limn→∞ 3/(5nln5) = 0

Оба условия признака Лейбница выполнены.


Пример 3 Исследовать сходимость ряда

∑∞n=0 (3n + 5)/(-2)n

Решение. Это знакочередующийся ряд. Выясним, выполняется ли первое условие признака Лейбница.

Так как un+1 = (3(n + 1) + 5)/2n+1 = (3n + 8)/2n+1, то

un - un+1 = (3n + 5)/2n - (3n + 8)/2n+1 = (6n + 10 - 3n - 8)/2n+1 = (3n + 2)/2n+1

Так как n ≥ 0, то 3n + 2 > 0. В свою очередь, 2n+1 > 0 для каждого n, поэтому (3n + 2)/2n+1. Следовательно, члены ряда по абсолютному значению убывают. Первый признак Лейбница выполняется. Выясним, равен ли нулю предел общего члена ряда (применяя правило Лопиталя???):

limn→∞ |(3n + 5)/(-2)n| = limn→∞ (3n + 5)/2n = ∞/∞ = limn→∞ 3/(2nln2) = 0

Получили нулевое значение. Оба условия признака Лейбница выполняются, поэтому данный ряд сходится.


Числовой ряд ∑∞k=1 ak называется абсолютно сходящимся, если сходится:

∑∞k=1 |ak|


Из абсолютной сходимости числового ряда следует его обычная сходимость (несколько упрощѐнно соотношение можно представить так: если сумма знакоположительно ряда конечна, то, поставив перед некоторыми его слагаемыми знак "-" , получим «перемешивание» слагаемых двух рядов – знакоположительного и знакоотрицательного, оба из которых сходятся.
